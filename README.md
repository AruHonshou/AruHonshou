## Proyectos Destacados

### [QAPilot](https://github.com/AruHonshou/QAPilot)

Plataforma local-first de QA con IA que convierte requerimientos funcionales en casos de prueba profesionales y automatizables.

**Stack:** React, TypeScript, Node.js, OpenAI API, Playwright, PostgreSQL, Docker, GitHub Actions

- Genera casos de prueba manuales o automatizados a partir de requerimientos en lenguaje natural.
- Crea planes declarativos de automatizacion UI/API sin ejecutar codigo arbitrario generado por IA.
- Ejecuta pruebas con Playwright, guarda evidencia por paso y genera reportes HTML, JSON y PDF.
- Incluye seguridad local-first: API key del usuario, cifrado, validacion con Zod, SSRF prevention y ejecucion en Docker.

---

### [DocuMente](https://github.com/AruHonshou/Documente)

Sistema RAG fullstack para cargar documentos, generar embeddings y responder preguntas con fuentes citables.

**Stack:** React, TypeScript, Node.js, OpenAI API, SQLite, Docker, GitHub Actions

- Procesa PDFs/TXT, divide documentos en chunks con overlap y recupera contexto por similitud coseno.
- Incluye RAGAgent con skills atomicas como busqueda, resumen, respuesta y extraccion de keywords.
- Implementa autenticacion JWT, streaming NDJSON, CI/CD y demo estatica en GitHub Pages.

---

### [OpenAI Chatbot Fullstack](https://github.com/AruHonshou/OpenAIBOTFrontend)

Aplicacion fullstack con frontend React y backend NestJS para interactuar con modelos de OpenAI.

**Repos:** [Frontend](https://github.com/AruHonshou/OpenAIBOTFrontend) | [Backend](https://github.com/AruHonshou/OpenAIBOTBackend)

**Stack:** React, TypeScript, TailwindCSS, NestJS, OpenAI API

- Chat conversacional con integracion directa a OpenAI API.
- Comparativa de modelos y soporte para vision/image-to-text.
- Separacion frontend/backend con arquitectura REST.

### [Infraestructura Automatizada con Terraform + AWS](https://github.com/AruHonshou/devops-terraform-project)

Proyecto DevOps con infraestructura como codigo, despliegue automatizado y app containerizada.

**Stack:** Terraform, AWS, Docker, GitHub Actions, Flask, Python

- Provisiona VPC, EC2, subnets, Security Groups, Elastic IP e IAM con minimos privilegios.
- Docker multi-stage para una Flask API servida con Gunicorn.
- Pipelines CI/CD para test, build, push, deploy y Terraform plan/apply.
